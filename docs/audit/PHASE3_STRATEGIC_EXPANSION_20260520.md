## Proposed Feature

### User Story

> As a **maker with multiple ideas**, I want the AI agents to understand my project domain and patterns from my past submissions, so that clarification questions and breakdowns are more relevant and personalized.

### Acceptance Criteria

1. When a clarification session starts, retrieve top 3 similar past ideas
2. Inject similar idea summaries into the clarifier prompt context
3. When breakdown runs, use similar completed ideas as templates
4. Add vector similarity score as confidence multiplier

### Value Justification

- **High Impact**: Personalized AI responses based on user history
- **Differentiation**: Competitors don't leverage user's own project corpus
- **Foundation**: Builds on existing pgvector infrastructure
- **Low Risk**: Uses existing similarity endpoints and embedding service

## Implementation Approach

1. Create `context-retrieval.ts` service in `/src/lib/agents/`
2. Modify clarifier to accept similarity context
3. Modify breakdown engine to use similar templates
4. Add tests for context retrieval
