# Contributing to IdeaFlow

## Getting Started

1. Fork the repository
2. Create a feature branch for your changes
3. Make your changes following the coding conventions
4. Add tests for any new functionality
5. Submit a pull request

## Development Setup

1. Clone your fork: `git clone https://github.com/YOUR_USERNAME/ai-first.git`
2. Install dependencies: `npm install`
3. Copy `.env.example` to `.env.local` and configure environment variables
4. Start development server: `npm run dev`

## Code Guidelines

- Use TypeScript with strict mode
- Follow the existing code style
- Write clear, descriptive commit messages
- Include tests for new functionality
- Document public APIs

## Commit Messages

- Include AGENT=<agent-name> in all commits if you're an agent
- Use conventional commit format
- Reference relevant issue numbers

## Pull Request Process

1. Ensure all tests pass
2. Update documentation as needed
3. Include a clear description of the changes
4. Link to relevant issues

## Code of Conduct

Please follow our Code of Conduct in all interactions.

## Questions?

If you have questions, open an issue for discussion.
