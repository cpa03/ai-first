# Skill Builder - Agentic QE

## Overview

This skill enables systematic skill building for AI agents with quality engineering principles. It provides a structured approach to creating, testing, and refining agent skills.

## When to Use

- Creating new agent skills
- Refactoring existing skills
- Testing skill effectiveness
- Building skill hierarchies

## Process

### 1. Skill Analysis

- Define skill purpose and scope
- Identify target agents and use cases
- Determine skill inputs and outputs
- Establish success criteria

### 2. Skill Design

- Write clear, actionable instructions
- Include examples and edge cases
- Define error handling strategies
- Create validation checkpoints

### 3. Skill Testing

- Test with various inputs
- Verify expected outputs
- Check edge case handling
- Validate against success criteria

### 4. Skill Refinement

- Analyze test results
- Iterate on instructions
- Improve clarity and effectiveness
- Document lessons learned

## Best Practices

- Keep skills focused and atomic
- Use clear, unambiguous language
- Include concrete examples
- Test thoroughly before deployment
- Version control skill changes
- Document dependencies and requirements

## Integration

Compatible with:

- OpenCode CLI
- Claude Code
- Gemini CLI

## Meta-Learning

This skill continuously improves itself by:

- Analyzing successful skill patterns
- Learning from user feedback
- Adapting to new agent capabilities
- Refining testing methodologies
